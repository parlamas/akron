2- 2-letter verb • irr- irregular • er- perfect with "være"
To find a verb, type -infinitive-, using either only English letters or Danish letters. For example, to find READ, you can type either -laese- or -læse- or -read-; to find UNDERSTAND, type -forsta- or -forstå- or -understand-; to find DO, type -gore- or -gøre- or -do-.

To find 2-letter verbs, type 2-; to find irregular verbs, type -irr-; to find regular verbs, type -reg-; to find verbs that take være in the perfect tenses, type er-.

To see which, if any, verbs start with the same 3 letters, type 3 first letters-; for example, bet- will filter out betyle and betale.
